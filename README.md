# 🧩 Logic Box Program

An interactive, terminal-based Python utility that combines creative geometric pattern generation with a robust numerical analysis engine. Built with advanced loop structures and dynamic control flows, this application features a styled ANSI-colored Command-Line Interface (CLI).

## ✨ Features

### 1. Pattern Generator (`case 1`)
Generates 4 custom geometric layouts based on user-defined row counts with vibrant terminal background coloring:
- **Right-Angle Triangle Pattern** (Magenta highlights)
- **Left-Angle Triangle Pattern** (Red highlights)
- **Inverted Pyramid Pattern** (Yellow highlights)
- **Standard Pyramid Pattern** (Black/Default highlights)

### 2. Number Analyzer (`case 2`)
Performs range-bound logical checks and aggregations:
- Classifies each integer within a specified range dynamically as **Even** or **Odd**.
- Leverages optimized conditional checks for swift classifications.
- Computes and prints the mathematical sum of the entire user-defined integer range using an iterative accumulation approach.

### 3. Styled Terminal UI
- Utilizes **ANSI escape codes** (`\033[...]`) for high-contrast color formatting, improving visibility and aesthetic layout structure directly inside terminal sessions.

---

## 🛠️ Python Concepts Covered
- **Control Flow Structures**: Implements Python’s modern `match-case` structural pattern matching paired with comprehensive nested `if-elif-else` logical layers.
- **Nested Iterations**: Complex layout rendering using layered `for` loops with calculated execution ranges.
- **Memory Optimization**: Efficient loop variables and variable manipulation (`while loop` accumulator pattern).
- **String Manipulations**: Custom formatting using keyword arguments (`end=" "`) inside print streams.
